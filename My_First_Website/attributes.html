Here are some important attributes I have come across. This is very far from an exhaustive list and isn't well organized yet. Still figuring out a good way to present this information.

*UNDER CONSTRUCTION*

a tag

  • href. This tells the anchor tag what it is linking to. This also works for download links, no difference.
    • ex: <a href=”page.htm”>

img tag

  • src. This tells the image tag what it is linking to. As far as I know it doesn't matter what kind of image it is.
    • ex: <img src=”iamage.png”>

img tag

  • src. This tells the image tag what it is linking to. As far as I know it doesn't matter what kind of image it is.
    • ex: <img src=”image.png”>
  • width and height. Defines how wide and how tall an image will be. You can use several differnt units for this such as percentages or pixels. It is smart too, if you only define a height or only a width it will scale the other appropriately
    • <img src=”image.jpg” width="500" height="600">
