Code Relationships

This is the basic relationship between the three core web development languages. HTML forms the base and is the minmimun requirement for what makes a website.

Even without any provided style and scripting pages will function with HTML alone. A page without CSS is not unstyled however, it will be styled using a default style provided by the browser the page is running in.

CSS provides the style, naturally. The look and feel of a website is provided by CSS. Websites without there own stylesheets, are simply put, unstylish.

Javascript provides something above and beyond style and structure. In short, Javascript can do anything but generally speaking Javascript is the final layer that takes the stuff provided by HTML and CSS and cuts it up, rearranges it and transforms it. Javascript can even add HTML and CSS information with script! This stuff is really magical.
